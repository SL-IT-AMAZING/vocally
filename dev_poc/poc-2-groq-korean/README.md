# PoC-2: Groq 한국어 정확도 테스트

## 검증 목표
Groq Whisper API의 한국어 음성 인식 품질 확인

## 가설
Groq Whisper large-v3가 한국어를 정확히 인식한다

## 성공 기준
- [ ] 30초 한국어 오디오 인식률 95%+
- [ ] 처리 시간 5초 이내
- [ ] 전문 용어(비즈니스) 인식 가능

## 사전 조건 (⏳ 사용자 작업 필요)

### Groq API 키 발급
1. https://console.groq.com 접속
2. 계정 생성
3. API Keys → Create API Key
4. 아래에 입력:
   - `GROQ_API_KEY`:

## 실행 방법

```bash
# 1. 환경변수 설정
cp .env.example .env
# .env 파일에 GROQ_API_KEY 입력

# 2. 테스트 실행
./test-groq-korean.sh
```

## 테스트 샘플

테스트에 사용할 한국어 문장:
1. 일반 회화: "안녕하세요, 오늘 회의 자료를 검토해 주세요."
2. 비즈니스 용어: "분기별 매출 보고서와 KPI 달성률을 확인하겠습니다."
3. 전문 용어: "API 엔드포인트 설정과 데이터베이스 마이그레이션을 진행합니다."
